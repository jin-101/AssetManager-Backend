## 3️⃣시스템 개요
### Asset Manager란
7가지 자산군(예적금, 자동차, 부동산, 주식, 코인, 외환, 금)에 대한 종합적인 자산관리 및 여러가지 통계 기능(개인 재무비율지표)을 제공해주는 하이브리드 앱입니다. 또한, 가계부 입력을 통해 연말정산 예상환급액 계산도 가능합니다.

### DB 구축을 위해 사용한 Open API 목록
|분류|기관|API 이름|
|:------|:------|:------|
|예적금|금융감독원|정기예금 API, 적금 API|
|부동산|국토교통부|아파트매매 실거래 상세 자료|
|자동차|국토교통부, KB캐피탈|자동차365, KB 차차차 (크롤링)|
|주식|한국거래소|유가증권 일별매매정보, 코스닥 일별매매정보|
|코인|업비트, 빗썸|현재가 정보 조회|
|외환|한국수출입은행|현재환율 API|
|금|금융위원회|일반상품시세정보|
